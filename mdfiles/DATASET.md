# SoccerAct10 - Football Action Dataset

This folder contains a collection of files intended for use as a **football action recognition and analysis dataset**.
The files are categorized by the type of event or action they represent on a football pitch.
SoccerAct10 is a dataset which contains 10 different soccer actions. This dataset was developed using the videos from YouTube.
